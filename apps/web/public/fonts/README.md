# WebDiag web fonts

- `manrope-ru-en-400-700.woff2` is a self-hosted subset generated from the supplied Manrope variable font.
- Included script coverage: English Latin, Russian Cyrillic, and punctuation/symbols used by the public RU/EN interface.
- Variable axis: `wght` restricted to `400..700`.
- Delivery: one WOFF2 file, preloaded by both root layouts.
- Original embedded license metadata: SIL Open Font License 1.1; copyright 2019–2021 The Manrope Project Authors.
- Code and machine-readable output continue to use the dedicated system monospace stack.

The four supplied static font files are intentionally not shipped because they duplicate the variable font and would increase the transferred font payload.
