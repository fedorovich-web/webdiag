# I18N.md — WebDiag

Дата: 2026-07-16
Статус: **accepted P0 requirement**

## Языки

- `ru` — основной язык;
- `en` — полноценный второй язык.

## URL model

- RU: `/`, `/tools/...`, `/blog/...`, `/guides/...`, `/glossary/...`;
- EN: `/en/`, `/en/tools/...`, `/en/blog/...`, `/en/guides/...`, `/en/glossary/...`;
- tool slugs одинаковы;
- переводы статей связаны через `translation_group_id`;
- canonical указывает на текущую языковую страницу;
- reciprocal hreflang: `ru`, `en`, `x-default`.

## Quality gates

- интерфейс, ошибки, findings, рекомендации и уведомления локализуются;
- машинный перевод без редакторской проверки не публикуется;
- язык переключается вручную;
- компоненты тестируются на расширение текста и переполнение.
